关于rUtil

date

date提供了获取当前时间戳和格式化指定日期的方法

示例代码

js

    var date = RU.date;
    var d = new Date();
    var ds = date.format(d,'yy-MM-dd'); //2014-05-03
    var dnow = date.now(); //返回当前时间戳，如：1403104207894
